## multi-text-form

### Built With

- HTML
- JQuery
- tailwindcss

## Installation

_Follow the steps below to run the project locally._

1. Clone the repo
   ```sh
   git clone https://github.com/AnthonyFranchitti/multi-text-form.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```

## Usage

1. Run BUILD (complies css)
   ```sh
   npm run build
   ```
2. Run DEV
   ```sh
   npm run dev
   ```
3. Run START (optional)
   ```sh
   npm run start
   ```
